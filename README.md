# Библиотека UI компонентов

Профессиональная библиотека React компонентов с темным дизайном для современных веб-приложений (обновляемая 2026)

!(https://i.postimg.cc/Bbrqy8pw/Snimok-ekrana-2026-01-19-v-01-48-04.png)

## Установка

```bash
npm install
```

## Запуск Storybook

```bash
npm run storybook
```

Откроется на `http://localhost:6006`

## Компоненты

- **Кнопка** - Основные, вторичные и кнопки удаления
- **Меню** - Контекстное меню с различными статусами
- **Поле ввода** - Текстовые поля, пароли, URL
- **Выпадающий список** - Селект с опциями
- **Переключатель** - Toggle switch
- **Карточка** - Селектируемые карточки (стандарт, премиум, королевская)
- **Тег** - Цветные теги с возможностью удаления
- **Счетчик** - Stepper для числовых значений
- **Чекбокс** - Флажки выбора
- **Ползунок** - Slider для диапазонов
- **Группа кнопок** - Сегментированный контрол
- **Переключатель темы** - Темная/светлая тема

## Дизайн

Все компоненты выполнены в темной цветовой схеме с акцентом на синий цвет (#5b6ef5) и плавными анимациями.
